Question-1: What are the different ways to select an element in the DOM?

There are different ways to select DOM elements. Some of the popular ways are:

getElementById ("id") : Selects by the element's unique ID.

getElementsByClassName ("class") : Returns a collection of elements with the specified class.

getElementsByTagName ("tag") : Returns elements with the given tag name .

querySelector ("CSS-selector") : Selects the first element matching a CSS selector.

querySelectorAll ("CSS-selector") : Returns a NodeList of all elements matching the CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML : Returns the HTML content (including tags) inside an element. Parses HTML.

innerText : Returns the visible text (rendered text, excluding hidden elements and tags).

textContent : Returns all text (including hidden text and line breaks) but ignores HTML tags.

Question-3: What is event delegation in the DOM?

Event Delegation is attaching an event listener to a parent element instead of individual child elements. Events from child elements bubble up to the parent, allowing us to handle them via event.target.

Benifits :-
Efficient for dynamic elements.
Reduces memory usage.

Question-4: What is event bubbling in the DOM?

Event Bubbling is the propagation of events upwards from the target element to its ancestors in the DOM tree.
For example, clicking a <button> triggers handlers on the button, its parent, grandparent, etc.

We Use event.stopPropagation() to stop bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating, Adding, and Removing Elements in JavaScript:

For Create an Element:
let newDiv = document.createElement("div");
newDiv.textContent = "Hello!";

Add an Element:
document.body.appendChild(newDiv);

Remove an Element:
newDiv.remove();
